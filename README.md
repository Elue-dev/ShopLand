# SHOPLAND (Full stack E-commerce application)

## Stacks: React, Redux toolkit, SASS, Firebase

### Third party libraries used (part of dependencies):
- React chatbot kit
- React chartjs
- React country region selector
- Redux
- React spinners
- React star rate
- React Toastify
- UUID
- Paystack
- Email JS
- Firebasse
- React stars rating


## Overview
An e-commerce shop with a variety of products from clothing to gadgets.
